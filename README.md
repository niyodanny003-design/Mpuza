# Mpuza UI

A fully responsive web-based user interface for the **Mpuza platform**, designed to manage jobs, professional networks, and company information. Built with **HTML**, **CSS**, and **Bootstrap Icons**, this UI showcases features such as user profiles, job postings, company pages, and professional groups.

## Overview

Mpuza UI is a prototype for a job and professional networking platform. It allows users to:

- Create and manage professional profiles
- View and follow company pages
- Post jobs and track applications
- Join and manage professional groups
- Monitor notifications, courses, and followers
- Access dashboards for company and professional metrics

The interface is divided into three main sections:

1. **Topbar** – navigation, search, notifications, job alerts, and profile actions.
2. **Sidebar** – user profile, metrics, company info, and professional groups.
3. **Main content area** – posts, job cards, featured companies, and job listings.
4. **Right panel** – favorited jobs, detailed job descriptions, and countdowns.

---

## Features

- **User Profile Dashboard:** Displays profile avatar, cover photo, headline, and personal metrics.
- **Company Pages:** Shows company info, followers, page visitors, and allows creating a new page.
- **Job Postings:** Post jobs, view applicants, and see professional stats.
- **Professional Groups:** Join or create groups for networking.
- **Notifications & Alerts:** Badge notifications for network, jobs, and courses.
- **Search Functionality:** Simple search bar for easy navigation.
- **Responsive Design:** Layout adjusts for desktop and mobile screens.
- **Interactive UI Components:** Buttons, badges, and icons enhance user experience.

## Technologies Used

- **HTML5** – for structuring the interface.
- **CSS3** – for styling and layout.
- **Bootstrap Icons** – for visual iconography.
- Optional: You can add **JavaScript** later for interactivity.
